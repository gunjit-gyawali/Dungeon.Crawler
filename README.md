# Dungeon.Crawler

Hey!
So this is my **first ever game** made using Godot 
It’s a simple 2D platformer where you run, jump, and try not to fall off stuff (which… will happen a lot).

---

## About the Game

This game is basically me learning how game development works.
You control a character that can:

* Move left and right
* Jump over gaps
* Avoid obstacles (or fail trying)

It’s not super polished or anything, but I’m proud of it because I actually made it from scratch.

---

## Goal

There’s no crazy story or anything (yet).
The goal is just to:

* Stay alive
* Reach the end of the level
* Not rage quit

---

## Controls

* **Arrow Keys** → Move and jump

(Simple, because I didn’t want to break everything on my first try)
